# 🏛️ ذاكرة ليبيا الرقمية — Libyan Digital Memory

منصّة رقمية تفاعلية (POC) لتوثيق المعالم التاريخية الليبية بتقنية **رمز الاستجابة السريعة QR** والوسائط الرقمية، مبنية على نموذج مشروع *Irish Statues* المرجعي، ومنفّذة كتطبيق **Angular 21**.

> «يكفي مسحُ رمزٍ لاستعادة آلاف السنين من التاريخ.»

هذه نسخة إثبات مفهوم (Proof of Concept) تعرض **10 معالم ليبية حقيقية** بداتا فعلية (إحداثيات، عصور، تاريخ موثّق بمصادره، وصور من Wikimedia Commons).

---

## ✨ المزايا

- **واجهة عربية أولاً (RTL)** مع تبديل فوري بين العربية والإنجليزية.
- **فهرس معالم** قابل للبحث والتصفية حسب المنطقة والعصر.
- **خريطة ليبيا تفاعلية** (Leaflet) بعلامات لكل معلم.
- **صفحة معلم متكاملة** تضمّ:
  - معرض صور متعدّد الزوايا.
  - **سرد صوتي حقيقي** للنبذة التاريخية عبر Web Speech API (دون استضافة ملفات).
  - **رمز QR مولّد ديناميكياً** يشير إلى رابط صفحة المعلم العامة.
  - خريطة Google Maps مدمجة + روابط Street View وفتح في الخرائط.
  - رابط فيديو وثائقي + بطاقة معلومات + المصادر والمراجع.
  - معالم ذات صلة + مشاركة.
- **نظام تصميم تراثي** (ألوان رملية/تيراكوتا/أزرق متوسطي/ذهبي، خطوط Amiri و Reem Kufi و Tajawal).
- **تحميل كسول (lazy loading)** لكل صفحة، و **fallback أنيق** للصور لا يُظهر صورة مكسورة أبداً.

## 🚀 التشغيل

```bash
npm install
npm start          # تطوير على http://localhost:4200
npm run build      # بناء إنتاجي في dist/
```

> **ملاحظة عن الصور:** روابط الصور من Wikimedia Commons وتُحمّل في متصفّح المستخدم. عند تعذّر تحميل أي صورة تظهر لوحة بديلة بهوية المشروع تلقائياً.

## 🧱 المعمارية

```
src/app/
├── core/
│   ├── models/         نماذج البيانات (Monument)
│   ├── data/           بيانات المعالم الحقيقية (monuments.data.ts)
│   ├── i18n/           قاموس الترجمة (ع/إن) + تسميات التصنيفات
│   └── services/       HeritageService · I18nService · QrService · SpeechService
├── shared/components/  smart-image · monument-card · site-header/footer · brand-mark
├── pages/              home · monuments · monument-detail · map · gallery · about
└── app.routes.ts       مسارات بتحميل كسول (/m/:slug لصفحة المعلم)
```

**التقنيات:** Angular 21 (standalone + signals + zoneless) · Tailwind CSS · Leaflet · qrcode · Web Speech API.

---

## 🔭 خريطة الطريق

- ربط الواجهة بـ **Backend (.NET / API)** بدل البيانات المحلية.
- لوحة تحكّم لإدارة المعالم ورفع الوسائط.
- دعم الإيطالية والفرنسية (البنية جاهزة).
- تسجيلات صوتية احترافية وفيديوهات توثيقية (إنتاج IZER).
- عرض ثلاثي الأبعاد 360° لكل معلم.
